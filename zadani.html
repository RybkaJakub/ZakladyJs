2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Ending statements with semicolon (;) is not required, but highly
  2. JavaScript ignores multiple spaces. You can add white space to your script to make it more readable. A good practice is to put spaces around oprators( = + - * /).
  3. For best readability, programmers often like to avoid code lines longer than 80 characters.
  4. JavaScript statements can be grouped together in code blocks, inside curly brackets (...). The prupose of code blocks is to define statements to be executed together.
  5. JavaScript keywords are reserved words. Reserved words cannot be used as names for variables.
  6. Sibgle line comments start with //. Multi-line comments start with /* and end with */.

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, *=, /=, %=, >>= ( posouvá proměnnou vpravo(se znaménkem)), <<=( posouvá proměnnou vlevo(se znaménkem)),>>= ( posouvá proměnnou vpravo(bez znaménka)), !=

Logické

!, || (operátor vrací true, pokud oba vyrazy jsou pravdivé) , && (operátor vrací true, pokud jsou oba výrazy správně vrátí true, jinak false)

Porovnávací

== , ===(stejná hodnota a stejný typ), !=, !==(nemá stejnou hodnotu nebo není stejný typ) , >, <, >=, <=

Bitové

&, |, ^ (Nastaví každý bit na 1, pokud je pouze jeden ze dvou bitů 1), ~(Invertuje všechny bity), >> (Posouvá doprava zatlačením kopií levého nejvíce levého bitu zleva a nechá vypadnout úplně pravé bity), << (Posouvá doleva zatlačením nul zprava a nechá vypadnout bity nejvíce vlevo), >>> (Posouvá doprava zatlačením nul zleva a nechá vypadnout bity nejvíce vpravo)

Typové

typeof (typ proměnné), instanceof (testování, zda je objekt daného typu)
